# URL Converter (YouTube â†’ MP3)

A simple desktop GUI tool to download audio from YouTube URLs and convert it to MP3.  
Built with **Python**, **Tkinter**, and **yt-dlp**.

---

## Features

- Download audio from YouTube URLs
- Convert to MP3 (192 kbps)
- Choose output folder
- Optional custom filename
- Optional track numbering (auto-increment)
---

## Requirements

- **Python 3.9 or newer**
- **FFmpeg** (required for MP3 conversion)
- Python package:
  - `yt-dlp`
